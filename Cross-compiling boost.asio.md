# Cross-compiling boost.asio

## Preparation
1. Download source from https://www.boost.org/
2. CMake (For demonstration)
3. ARM-GCC

## Build
1. Unzip the source.
`tar --bzip2 -xf boost_1_67_0.tar.bz2 && cd boost_1_67_0` 
2. enter the source folder.
